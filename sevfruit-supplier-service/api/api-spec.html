Сервис учёта поставок (1.0)

Download OpenAPI specification:Download

License: Unlicense

API сервиса учёта поставок. Используется для внесения информации о поставщиках, продукции, ценах на продукцию на каждый период поставок и поставках продукции. Предусмотрено создание отчёта о поставках продукции поставщиками в заданном периоде.

Авторизация

Для определения прав доступа к методам API используется авторизация API key. Ключ доступа передаётся в заголовке Api-Key.

Ошибки

API использует стандартные HTTP коды для сообщения об успешности или не успешности вызова API. В случае неуспеха, теле ответа содержится JSON с информацией об ошибке.
